# Script-Shereck
# Envio automatizado de mensagens no WhatsApp Web

Este script permite o envio automatizado de múltiplas mensagens em uma conversa do WhatsApp Web, utilizando JavaScript executado diretamente no console do navegador. Cada linha de texto informada é enviada como uma mensagem individual.

⚠️ Este projeto é destinado a fins educacionais e de automação controlada. O uso indevido pode violar os termos de uso do WhatsApp.

---

## Pré-requisitos

- Navegador compatível (Google Chrome, Microsoft Edge ou similares)
- Acesso ao WhatsApp Web
- Conversa aberta e totalmente carregada

---

## Instruções de uso

1. Acesse o WhatsApp Web pelo navegador.
2. Selecione e abra a conversa (contato ou grupo) onde as mensagens serão enviadas.
3. Abra as ferramentas de desenvolvedor do navegador:
   - Pressione `F12` ou
   - Clique com o botão direito na página e selecione **Inspecionar**
4. Acesse a aba **Console**.
5. Cole o conteúdo do script JavaScript e pressione **Enter** para registrá-lo.
6. Execute a função `enviarScript()` passando o texto desejado como parâmetro.

Cada linha do texto informado será enviada como uma mensagem separada na conversa aberta.

---

## Observações

- O script depende da estrutura atual do WhatsApp Web, que pode mudar sem aviso.
- Recomenda-se utilizar intervalos de envio adequados para evitar bloqueios ou instabilidade.
- Não utilize para envio de spam ou mensagens em massa não autorizadas.
